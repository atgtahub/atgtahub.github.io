## 使用n管理nodejs版本

安装
```sh
npm install -g n
```

## 管理node

- sudo n ls-remote: 列出最新版本列表
- sudo n ls-remote 16: 列出16x的版本
- sudo n stable: 安装最新稳定版
- sudo n latest: 安装最新版本
- sudo n 16.20.2: 安装指定版本
- sudo n rm 16.20.2: 删除指定版本
- sudo n ls: 查看本地版本列表
- sudo n: 切换版本
